# CoffeeCorp_Network_Security Project
This project demonstrates the design and security of a small enterprise (CoffeeCorp) network
using Cisco Packet Tracer.

## Features
- VLAN segmentation (Sales, Finance, Servers)
- Inter-VLAN routing using router-on-a-stick
- Access Control Lists (ACLs) to protect the File Server
- DNS configuration and connectivity testing

## Files
- `/packet_tracer` – Packet Tracer project file (.pkt)
- `/screenshots` – Verification screenshots
- `/report` – Final project report (PDF)
